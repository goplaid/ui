# vuetifyjs

## Project setup
```
cd goplaid/web/corejs
npm run serve

cd goplaid/ui/vuetifyx/vuetifyxjs
npm run serve

cd goplaid/ui/vuetifyx/vuetifyxjs
python -m SimpleHTTPServer

or

go install github.com/sunfmin/branfileserver@master
cd goplaid/ui/vuetifyx/vuetifyxjs
branfileserver

```
